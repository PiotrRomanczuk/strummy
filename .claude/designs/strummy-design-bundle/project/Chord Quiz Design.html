In-quiz question

The chord diagram is the protagonist — large, centered, on a flat card with one faint gold corner-glow to feel premium without sparkle-spam. Answers sit in a 2×2 grid on mobile and a stacked single column on desktop, each prefixed with a keystroke hint (1–4) so power users never have to lift hands from the keys. The progress bar replaces the page header so the eye lands on the chord, not on chrome. A single heart and streak icon live on the top-right — visible but never shouting.

// components/skills/ChordQuiz/ChordQuiz.Question.tsx
'use client';

import { motion, AnimatePresence } from 'framer-motion';
import { Heart, Flame, X, Eye, ChevronRight, Volume2, Clock } from 'lucide-react';
import { Button } from '@/components/ui/button';
import { Progress } from '@/components/ui/progress';
import { ChordDiagram } from './ChordDiagram';
import { cn } from '@/lib/utils';
import { type QuizQuestion } from './chord-quiz.helpers';

interface Props {
  question: QuizQuestion;
  questionNumber: number;
  totalQuestions: number;
  hearts: number;
  streak: number;
  selected: string | null;
  onSelect: (option: string) => void;
  onClose: () => void;
}

export function ChordQuizQuestion({
  question, questionNumber, totalQuestions, hearts, streak, selected, onSelect, onClose,
}: Props) {
  const progressPct = (questionNumber / totalQuestions) * 100;

  return (
    <div className="flex flex-col bg-background text-foreground">
      {/* Top bar: close · progress · hearts · streak */}
      <header className="flex items-center gap-3 px-4 lg:px-10 pt-3 pb-3 border-b border-border/50">
        <Button variant="ghost" size="icon" onClick={onClose} aria-label="Close quiz">
          <X className="h-5 w-5" />
        </Button>
        <Progress value={progressPct} className="flex-1 h-2.5" />
        <div className="flex items-center gap-1.5 text-rose-500">
          <Heart className="h-5 w-5 fill-rose-500" />
          <span className="font-semibold tabular-nums text-foreground">{hearts}</span>
        </div>
        <div className="flex items-center gap-1.5 text-amber-500">
          <Flame className="h-5 w-5 fill-amber-500" />
          <span className="font-semibold tabular-nums text-foreground">{streak}</span>
        </div>
      </header>

      {/* Body — single column on mobile, 2-col on lg+ */}
      <div className="px-4 py-6 lg:max-w-5xl lg:mx-auto lg:px-10 lg:py-12 lg:grid lg:grid-cols-2 lg:gap-16 lg:items-center">
        <section className="space-y-5">
          <p className="text-[11px] font-medium uppercase tracking-[0.14em] text-muted-foreground">
            Question {questionNumber} of {totalQuestions} · Name the chord
          </p>
          <h1 className="font-display text-[26px] lg:text-4xl font-semibold tracking-tight">
            Which chord is this?
          </h1>

          <div className="relative rounded-2xl border bg-card p-5 lg:p-8 overflow-hidden">
            <div className="pointer-events-none absolute -top-12 -right-12 w-40 h-40 rounded-full bg-primary/10 blur-2xl" />
            <ChordDiagram voicing={question.voicing} size="xl" hideName />

            <Button variant="secondary" size="icon" aria-label="Play chord audio"
              className="absolute bottom-4 right-4 h-11 w-11 rounded-full">
              <Volume2 className="h-5 w-5" />
            </Button>
          </div>
        </section>

        <section className="mt-6 lg:mt-0 space-y-3 lg:space-y-4">
          <div className="grid grid-cols-2 lg:grid-cols-1 gap-3">
            {question.options.map((opt, i) => (
              <AnswerChoice
                key={opt}
                index={i}
                label={opt}
                selected={selected === opt}
                onClick={() => onSelect(opt)}
              />
            ))}
          </div>
          <div className="flex items-center justify-between pt-2">
            <button className="text-xs text-muted-foreground hover:text-foreground inline-flex items-center gap-1.5">
              <Eye className="h-3.5 w-3.5" /> Use a hint
            </button>
            <button className="text-xs text-muted-foreground hover:text-foreground inline-flex items-center gap-1.5">
              Skip <ChevronRight className="h-3.5 w-3.5" />
            </button>
          </div>
        </section>
      </div>
    </div>
  );
}

// 1.6:1 tile on mobile, full-width row on desktop. Keystroke hint top-left.
function AnswerChoice({ index, label, selected, onClick }: {
  index: number; label: string; selected: boolean; onClick: () => void;
}) {
  const key = ['1', '2', '3', '4'][index];
  return (
    <button
      onClick={onClick}
      aria-pressed={selected}
      className={cn(
        'group rounded-xl border border-border bg-card hover:border-primary/60 hover:bg-secondary/40 transition-colors',
        'aspect-[1.6] lg:aspect-auto grid place-items-center relative',
        'lg:flex lg:items-center lg:gap-4 lg:px-5 lg:py-4 lg:text-left lg:grid-cols-[auto_1fr_auto]',
      )}
    >
      <span className="absolute top-2 left-2 font-mono text-[10px] text-muted-foreground lg:static lg:h-8 lg:w-8 lg:rounded-md lg:border lg:border-border lg:bg-secondary lg:grid lg:place-items-center lg:text-xs">
        {key}
      </span>
      <span className="font-display text-2xl lg:text-xl font-semibold tracking-tight">{label}</span>
      <ChevronRight className="hidden lg:block h-4 w-4 ml-auto text-muted-foreground group-hover:text-primary" />
    </button>
  );
}

Quiz home

Greets students by name and answers three questions at a glance: am I on a streak, how much have I done today, and what should I play next? Mode cards lead with the workhorse — Name the Chord — given the strongest visual weight (gold border + glow), with the audio and build modes rendered as equal peers below. A small "Working on" rail at the bottom previews chords the student is struggling with, pulled from the same mastery model the teacher dashboard already uses.

  • · Name personalisation ("Good evening, Maya") instead of a static "Welcome back" — pulled from getUserWithRolesSSR.
  • · One emphasized primary mode; the others equal. Avoids the 3-equal-cards-with-3-different-colors pattern.
  • · "Working on" rail uses real chord diagrams as cards — not generic colored tiles.
  • · Daily-goal copy is concrete: "12 XP to keep your streak" — not "Complete daily goal."
  • · Stat tiles use icon + label + value, no donut charts. Density without noise.
  • · Avg. response time shown as a stat — desktop only, surfaces what the algorithm cares about.

Correct / incorrect overlay

Bottom sheet on mobile, centred dialog on desktop. The colour comes through a single top border on the sheet, not a saturated full background — the chord itself stays the focus. Both states show the answer, the diagram, the audio button, and a one-line tip tying this chord to one the student already knows ("Dm7 shares a shape with Dm…"). The Continue CTA is full-width and confirmed with a Space-bar hint so a streak run never breaks rhythm.

→ Incorrect state

  • · Bottom-sheet, not modal-takeover on mobile — feedback feels native, the prior question is still visible.
  • · Single coloured top border (emerald or rose, 2px) for state — not a tinted background.
  • · The tip line ties this chord to a known one — pedagogically useful, not generic encouragement.
  • · Audio replay anchored to the answer, not a separate "Listen again" button.
  • · "You picked Am7" shown explicitly on incorrect — students need to see their mistake.
  • · Spacebar shortcut visible so power users can chain answers without a mouse.

Session summary

The accuracy number is enormous and serif — this is the dopamine moment. Beneath it: the question-by-question dot strip (a small honest receipt), then two mastery cards showing chords moving up and chords moving down. The "next quiz in" timer creates the spaced-repetition habit without nagging — and the Notify-me opt-in is right there, not buried in settings.

  • · Accuracy as oversized serif numeral — Fraunces at 140px, hits like an album cover.
  • · Per-question receipt (12 coloured dots) — students see exactly where they slipped.
  • · Up/down mastery cards — both directions celebrated, not just wins.
  • · "Next quiz in 3h 24m" reinforces spaced repetition without a streak guilt-trip.
  • · Notify-me inline opt-in turns this screen into a retention loop.
  • · No confetti, no mascot. The gold number does all the celebrating.
