Totals: 10 new test files, 86 new test cases, all passing.

Skipped by design: the plan proposed two optional integration tests (getAtRiskStudents, IntegrationsSection's disconnect flow) hitting the real dev Supabase. On inspection, this repo's *.integration.test.ts suite actually mocks Supabase (via lib/testing/integration-helpers.ts) to test cross-module business logic, and the only suite that hits a real dev-branch DB is the separate *.rls.test.ts harness, scoped specifically to row-level-security policy assertions. Neither convention fit what was proposed, so — rather than invent a third pattern — these were skipped. Flagging here instead of silently dropping them.
