## Journey 15: AI Playground — Chat

**Priority**: P1
**Role**: Admin/Teacher
**Pages**: `/dashboard/ai`, `/dashboard/ai/history`
**Existing coverage**: Partial (ai-playground.spec.ts)

### Preconditions
- Logged in as admin or teacher
- AI provider configured (OpenRouter or Ollama)

### Happy Path

#### Step 1 — Open AI Playground
1. Navigate to `/dashboard/ai`
2. **Expect**: "AI Assistant Playground" page loads
3. **Expect**: Welcome message / system greeting
4. **Expect**: Chat input visible with placeholder text
5. **Expect**: Model selector dropdown (shows available models)
6. **Expect**: Send button disabled when input is empty

#### Step 2 — Send a message
1. Type: `What are some good beginner guitar songs?`
2. Click Send (or press Enter)
3. **Expect**: User message appears in the chat
4. **Expect**: Input field clears
5. **Expect**: Loading/typing indicator appears
6. **Expect**: AI response streams in within 30s
7. **Expect**: Response contains relevant content about guitar songs

#### Step 3 — Continue the conversation
1. Type: `Can you give me chord progressions for those songs?`
2. Send
3. **Expect**: AI responds with context from previous message (conversation memory)

#### Step 4 — Change model
1. Open model selector dropdown
2. Select a different model
3. Send a new message
4. **Expect**: Response generated by the new model

#### Step 5 — Clear conversation
1. Click "Clear" / "New Chat" button
2. **Expect**: Chat history cleared
3. **Expect**: Welcome message returns

#### Step 6 — View conversation history
1. Navigate to `/dashboard/ai/history`
2. **Expect**: Previous conversation(s) listed
3. Click on a conversation
4. **Expect**: Full message history displayed

### Edge Cases

#### E1 — Empty message
1. Click Send with empty input
2. **Expect**: Nothing happens (button disabled or no-op)

#### E2 — Very long message
1. Type 5000+ characters
2. **Expect**: Either accepted or truncated with warning

#### E3 — Suggested prompts
1. On the welcome screen, click one of the suggested prompts
2. **Expect**: Prompt auto-fills and sends, AI responds

- Delete test AI conversations via DB

## Journey 16: AI Lesson Notes Generation

**Priority**: P1
**Role**: Teacher
**Pages**: `/dashboard/lessons/new` or `/dashboard/lessons/[id]/edit`
**Existing coverage**: Partial (lesson-notes-ai.spec.ts)

### Preconditions
- Logged in as teacher
- At least one student and one song exist

### Happy Path

#### Step 1 — Open lesson form
1. Navigate to `/dashboard/lessons/new`
2. **Expect**: Form loads with student, date, title, notes fields

#### Step 2 — Configure context for AI
1. Select a student from dropdown
2. Enter title: `Guitar Lesson - Fingerpicking`
3. Select 1-2 songs from the song selector
4. **Expect**: "Generate Notes with AI" button becomes visible/enabled

#### Step 3 — Generate AI notes
1. Click "Generate Notes with AI"
2. **Expect**: Button shows "Generating..." state
3. **Expect**: Loading spinner or streaming indicator
4. **Expect**: Within 30s, notes field populates with structured content
5. **Expect**: Generated notes reference the selected songs and student context

#### Step 4 — Edit generated notes
1. Modify the AI-generated notes (add a personal note)
2. Save the lesson
3. **Expect**: Lesson saved with the modified AI notes

### Edge Cases

#### E1 — AI button hidden without required context
1. Don't select a student or songs
2. **Expect**: AI button is NOT visible or is disabled

#### E2 — AI generation fails
1. If AI provider is down, click Generate
2. **Expect**: Error message (not a crash)
3. **Expect**: Notes field remains editable (user can type manually)

### Cleanup
- Delete test lesson

## Journey 18: AI Email Draft Generation

**Priority**: P2
**Role**: Teacher/Admin
**Pages**: Email draft generation UI (within student detail or standalone)
**Existing coverage**: None

### Preconditions
- Logged in as teacher
- At least one student exists with some lesson/assignment history

### Happy Path

#### Step 1 — Access email draft generator
1. Navigate to the email draft feature (e.g., from student detail page action menu, or `/dashboard/ai`)
2. **Expect**: Template type selector with 4 options:
   - Lesson reminder
   - Progress report
   - Payment reminder
   - Milestone celebration

#### Step 2 — Generate lesson reminder email
1. Select "Lesson reminder" template
2. Select a student
3. Click "Generate"
4. **Expect**: AI generates a professional email draft
5. **Expect**: Draft includes student name, lesson date/time, preparation notes

#### Step 3 — Generate progress report email
1. Select "Progress report" template
2. Select the same student
3. Click "Generate"
4. **Expect**: AI generates a progress summary referencing the student's recent lessons, song progress

#### Step 4 — Edit and copy/send draft
1. Review the generated draft
2. Edit the subject or body
3. Copy to clipboard or send (document available actions)
4. **Expect**: Draft is usable for actual sending

### Edge Cases

#### E1 — Student with no history
1. Select a new student with no lessons or songs
2. Generate a progress report
3. **Expect**: AI handles gracefully (generic report or message about insufficient data)

### Cleanup
- None needed (drafts are not persisted unless sent)
