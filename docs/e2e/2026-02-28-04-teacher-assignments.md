## Journey 8: Assignment CRUD

**Priority**: P0
**Role**: Teacher
**Pages**: `/dashboard/assignments`, `/dashboard/assignments/new`, `/dashboard/assignments/[id]`, `/dashboard/assignments/[id]/edit`
**Existing coverage**: Partial (teacher-full-journey.spec.ts)

### Preconditions
- Logged in as teacher
- At least one student and one song exist

### Happy Path

#### Step 1 — Navigate to assignments
1. Click "Assignments" in sidebar
2. **Expect**: `/dashboard/assignments` loads
3. **Expect**: Assignment list with columns: title, student, status, due date
4. **Expect**: "New Assignment" button visible
5. **Expect**: Status filter (not_started, in_progress, completed, overdue)

#### Step 2 — Create assignment
1. Click "New Assignment"
2. **Expect**: Form loads with fields: title, description, student, due date, linked song, linked lesson
3. Fill in:
   - Title: `E2E Assignment {timestamp}`
   - Description: `Practice chord transitions for 15 minutes daily`
   - Student: select from dropdown
   - Due date: 1 week from today
   - Song: select a song (optional)
4. Click "Save"
5. **Expect**: Redirect to assignment detail
6. **Expect**: Status shows "not_started"
7. **Expect**: Due date formatted correctly

#### Step 3 — View assignment detail
1. On assignment detail page
2. **Expect**: Title, description, student name, due date, status all visible
3. **Expect**: Linked song shown (if selected)
4. **Expect**: "Edit" and "Delete" buttons visible for teacher

#### Step 4 — Edit assignment
1. Click "Edit"
2. Change description to `Updated: Practice 20 minutes daily`
3. Change due date to 2 weeks from today
4. Click "Save"
5. **Expect**: Updated values reflected on detail page

#### Step 5 — Delete assignment
1. Click "Delete" -> Confirm
2. **Expect**: Redirect to assignments list
3. **Expect**: Assignment removed from list

### Edge Cases

#### E1 — Create without required fields
1. Leave title empty, click Save
2. **Expect**: Validation error on title

#### E2 — Due date in the past
1. Set due date to yesterday
2. **Expect**: Either warning or error — document behavior

#### E3 — Assignment overdue auto-detection
1. Create an assignment with due date = today
2. Wait for the cron job `assignment-overdue-check` to run (or trigger manually)
3. **Expect**: Assignment status changes to "overdue" automatically
4. **Expect**: Overdue badge/indicator on the assignment in the list

### Cleanup
- Delete `E2E Assignment {timestamp}`

## Journey 9: Assignment Templates

**Priority**: P2
**Role**: Teacher
**Pages**: `/dashboard/assignments/templates`, `/dashboard/assignments/templates/new`, `/dashboard/assignments/templates/[id]`
**Existing coverage**: None

### Preconditions
- Logged in as teacher

### Happy Path

#### Step 1 — Navigate to templates
1. Go to `/dashboard/assignments/templates`
2. **Expect**: Template list page loads
3. **Expect**: "New Template" button visible

#### Step 2 — Create a template
1. Click "New Template"
2. Fill in: title (`E2E Template {timestamp}`), description (`Weekly practice routine template`)
3. Click "Save"
4. **Expect**: Template appears in the list

#### Step 3 — Use template to create assignment
1. Navigate to `/dashboard/assignments/new`
2. Select the template from a template dropdown/selector (if available in the create form)
3. **Expect**: Title and description pre-fill from the template
4. Select a student and due date
5. Click "Save"
6. **Expect**: Assignment created with template's content

#### Step 4 — Delete template
1. Navigate back to templates list
2. Delete the test template
3. **Expect**: Template removed from list

### Edge Cases

#### E1 — Teacher can only see own templates
1. As Teacher A, create a template
2. Log in as Teacher B (if a second teacher account exists)
3. View templates list
4. **Expect**: Teacher A's template is NOT visible to Teacher B

### Cleanup
- Delete `E2E Template {timestamp}` and associated assignment
