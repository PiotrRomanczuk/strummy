## Journey 19: Student Data Export (PDF/XLSX)

**Priority**: P1
**Role**: Teacher/Admin
**Pages**: `/dashboard/users/[id]` (export tab/section)
**Existing coverage**: None

### Preconditions
- Logged in as teacher
- A student exists with lessons, songs in repertoire, and assignments

### Happy Path

#### Step 1 — Navigate to student detail
1. Go to `/dashboard/users/[student-id]`
2. Click "Export" tab or find export button
3. **Expect**: Export options visible: PDF, XLSX (Excel)

#### Step 2 — Export as PDF
1. Click "Export PDF"
2. **Expect**: PDF download triggers (or opens in new tab)
3. **Expect**: PDF contains:
   - Student name and profile info
   - Lesson history (dates, songs covered, statuses)
   - Repertoire list with progress statuses
   - Assignment summary

#### Step 3 — Export as XLSX
1. Click "Export XLSX" / "Export Excel"
2. **Expect**: XLSX file downloads
3. **Expect**: Spreadsheet contains structured data:
   - Sheet 1: Student info
   - Sheet 2: Lessons (date, title, status, songs)
   - Sheet 3: Repertoire (song, status, rating)
   - Sheet 4: Assignments (title, status, due date)

### Edge Cases

#### E1 — Student with no data
1. Export a newly created student with no lessons/songs/assignments
2. **Expect**: PDF/XLSX still generates but with empty sections (not an error)

#### E2 — Large dataset
1. Export a student with 100+ lessons
2. **Expect**: Export completes without timeout (may take a few seconds)

### Cleanup
- None (downloaded files)
